# Extrofreight

Marketing website for Extrofreight — a logistics company offering nationwide
freight & logistics services, with home moving as its flagship service.

Built with Next.js (App Router), TypeScript and Tailwind CSS v4.

## Pages

| URL | Description |
|-----|-------------|
| `/` | Home — overview of all services |
| `/home-moving` | Flagship home moving service: move types, pricing, process, FAQ |
| `/services` | Freight (LTL/FTL), office relocation, warehousing & storage |
| `/quote` | Free quote request form |
| `/track` | Shipment tracking |
| `/about` | Company story, mission & values |
| `/contact` | Contact details & message form |

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
